System Architecture Layers

Frontend Layer
Vanilla JS ES6 Modules Leaflet
  • main.js - Application entry point
  • api-client.js - HTTP with auto X-Client-ID
  • map-manager.js - Folium iframe control
  • ui-manager.js - DOM, modals, tabs
  • message-manager.js - Chat UI
  • simulation-manager.js - WTA polling
  • + 4 more modules
Backend Layer
Flask Python Folium
  • 8 Route Blueprints (22+ endpoints)
  • llm_service.py - Ollama integration
  • map_service.py - Map drawing logic
  • map_state.py - Session state model
  • fallback_handler.py - Regex fallback
  • parser.py - LLM output parsing
  • helpers.py - Session utilities
LLM Layer
Ollama llama3.2:3b Function Calling
  • Intent understanding
  • Parameter extraction
  • import_scenario tool
  • get_wta tool
  • get_track tool
  • text_generation tool
  • Configurable prompts
Data Layer
JSON Local Files Per-Session
  • db_v2.json - Ship database
  • track_data.json - Trajectory waypoints
  • config.json - Runtime switches
  • prompts_config.json - LLM prompts
  • maps/ - Generated HTML files
  • feedbacks/ - User feedback
External Services
Node.js API Port 3000 REST
  • GET /api/v1/ships - Ship coordinates
  • POST /api/v1/wta - Weapon allocation
  • GET /api/v1/tracks - Trajectory data
  • Supports local/mock/real modes
  • Automatic mode switching via config

Key API Endpoints

Scenario Management
POST /api/import_scenario Draw ships
POST /api/start_scenario Launch simulation
POST /api/clear_map Clear all layers
Data Queries
POST /api/get_wta Weapon allocation
POST /api/get_track Draw trajectories
GET /api/check_simulation_status/:id Poll status
Military Q&A
POST /api/get_answer RAG-based Q&A
Configuration
GET /api/prompts/list List configs
POST /api/prompts/update/:name Update prompts
GET /health Health check

Deployment

Prerequisites
  • Python 3.8+
  • Ollama with llama3.2:3b
  • Node.js backend (optional)
  • Modern browser (Chrome 61+)
Port Configuration
  • Flask: 5000
  • Ollama: 11434
  • Node.js API: 3000
API Modes
  • local - JSON files
  • mock - Simulated responses
  • real - Node.js/external API
